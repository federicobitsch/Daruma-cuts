#  Daruma Cuts - Landing Page 

¡Bienvenido al repositorio de **Daruma Cuts**! Este es mi primer proyecto profesional como desarrollador de software, diseñado específicamente para una barbería de autor en la ciudad de **Río Grande, Tierra del Fuego**.


El objetivo de este proyecto es ofrecer una experiencia de usuario rápida, moderna y 100% orientada a la conversión de clientes mediante reserva directa.

---
## 🚀 Tecnologías Utilizadas

Este proyecto fue migrado de una estructura estática a un entorno moderno de alto rendimiento:

* **Frontend:** [React.js](https://reactjs.org/) & [Next.js 15](https://nextjs.org/) (App Router).
* **Lenguaje:** [TypeScript](https://www.typescript.org/) para un código más robusto y escalable.
* **Estilos:** [Bootstrap 5](https://getbootstrap.com/) & [Tailwind CSS](https://tailwindcss.com/) para un diseño responsivo y "Mobile First".
* **Iconos:** [Bootstrap Icons](https://icons.getbootstrap.com/).
* **Despliegue:** [Vercel](https://vercel.com/) para carga instantánea y optimización de imágenes.

---

## ✨ Características Principales

* ✅ **Diseño Dark Premium:** Estética elegante en negro y rojo, ideal para el rubro de barbería.
* ✅ **Sistema de Turnos por WhatsApp:** Formulario inteligente que envía un mensaje pre-formateado al barbero con el nombre, servicio y fecha elegida.
* ✅ **Validación de Fechas:** Lógica en TypeScript para impedir reservas en fechas pasadas.
* ✅ **Mapa Interactivo:** Integración de Google Maps con filtros CSS para combinar con el modo oscuro.
* ✅ **100% Responsivo:** Optimizado para que los clientes reserven cómodamente desde su celular.
* ✅ **Animaciones de Scroll:** Efectos de "Fade-in" al navegar por las secciones.
